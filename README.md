# Hardware Guides

This repository contains my beginner-friendly hardware guides for Hack Club’s [Resolution program](https://resolution.hackclub.com/). 

![](attachments/zen_rF6zhkHyEQ.jpg)


Each guide walks you step-by-step through a weekly project, making it easy to learn hardware concepts and build real projects, even if you’re new to electronics.
